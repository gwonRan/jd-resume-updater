# Resume JD Updater

`resume-jd-updater`는 목표 JD와 프로젝트 메모를 바탕으로 이력서와 경력기술서를 업데이트하는 Codex skill입니다.

## 주요 기능

- JD 분석
- 마스터 이력서 재료집 생성
- JD-경험 매칭
- 성과 언어 변환
- 스킬/도구 맥락화
- 기간/기한 반영
- 이력서 bullet 및 섹션 업데이트
- 후속 질문 생성

## 사용 예시

```text
Use $resume-jd-updater to update my resume for this JD.

목표 JD:
...

프로젝트 메모:
...
```
