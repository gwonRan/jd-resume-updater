# 이력서 Bullet 작성 패턴

## 기본 공식

`문제/목표 + 내 행동 + 방법/도구 + 결과`

또는:

`Action + scope/context + method/tool + result`

## 패턴

### 성과 중심

`[방법/도구]를 통해 [지표/프로세스]를 [결과]만큼 개선`

### 오너십 중심

`[업무 영역]을 담당하고 [이해관계자]와 협업해 [산출물/결과]를 달성`

### 문제 해결 중심

`[문제]를 [분석/실행/의사결정]으로 해결해 [결과]에 기여`

### 협업 중심

`[팀/이해관계자]와 협업해 [요구사항/트레이드오프]를 조율하고 [결과]를 도출`

### 수치 없는 버전

`[목표]를 지원하기 위해 [프로세스/산출물]을 구축하고 [팀/고객/운영]의 실행 기반을 강화`

## 주의

- "리드", "총괄", "고도화"는 근거가 있을 때만 쓴다.
- 팀 성과는 팀 성과로 쓰고, 개인 기여가 명확할 때만 개인 성과로 쓴다.
- 성과 수치가 없으면 `[확인 필요]`로 표시하거나 정성 성과로 작성한다.
