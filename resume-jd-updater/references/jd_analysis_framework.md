# JD 분석 프레임워크

## 추출 항목

- 직무명과 예상 레벨
- 핵심 책임
- 필수 역량
- 우대 역량
- 반복 키워드
- 도구/방법론/도메인
- 협업과 이해관계자 기대사항
- 리더십/오너십 신호
- 이 역할이 해결해야 하는 비즈니스 문제

## 분류

- **반드시 보여줄 것**: 서류 평가에서 핵심일 가능성이 높은 요구사항
- **차별화 요소**: 보여주면 강점이 되는 요구사항
- **가볍게 반영**: 언급은 가능하지만 핵심은 아닌 요구사항
- **공백/리스크**: 현재 자료로는 근거가 부족한 요구사항
