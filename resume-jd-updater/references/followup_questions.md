# 후속 질문

## 질문 우선순위

- **P1**: 없으면 문장을 쓰기 어렵거나 사실 리스크가 큰 정보
- **P2**: 있으면 성과와 설득력이 강해지는 정보
- **P3**: 있으면 포지셔닝과 표현이 좋아지는 정보

## 우선 질문

- P1: 이 프로젝트에서 본인이 직접 오너십을 가진 범위는 어디까지였나요?
- P1: 팀 성과와 개인 기여를 어떻게 구분할 수 있나요?
- P1: 외부 공개하면 안 되는 고객명, 제품명, 수치, 내부 시스템명이 있나요?
- P2: 성과를 전/후, 목표 대비, 증가/감소율로 표현할 수 있나요?
- P2: 프로젝트 기간, 출시일, 마감일, 운영 기간이 있나요?
- P2: 사용한 스킬/도구가 실제로 어떤 결과를 가능하게 했나요?
- P3: 목표 JD에서 가장 강조하고 싶은 역량은 무엇인가요?

## 질문 방식

질문은 3-7개로 제한한다. 이미 제공된 자료에서 알 수 있는 내용은 다시 묻지 않는다.
