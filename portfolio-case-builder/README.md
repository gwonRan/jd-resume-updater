# Portfolio Case Builder

`portfolio-case-builder`는 프로젝트 메모를 채용용 포트폴리오 케이스 스터디로 바꾸는 Codex skill입니다.

## 주요 기능

- 포트폴리오 목차 구성
- 프로젝트 증거 추출
- 문제 정의-해결 과정-결과-회고 구조화
- 스킬/도구 활용 맥락 정리
- 성과 시각화 아이디어 도출
- JD와 연결되는 역량 매핑
- 후속 질문 생성

## 사용 예시

```text
Use $portfolio-case-builder to create portfolio case studies from these project notes.

목표 포지션/JD:
...

프로젝트 메모:
...
```
