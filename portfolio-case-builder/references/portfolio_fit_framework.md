# 포트폴리오 적합도 판단 프레임워크

모든 프로젝트가 포트폴리오에 적합한 것은 아니다. 케이스 스터디로 확장하기 전에 포트폴리오 적합도를 판단한다.

## 평가 기준

| 기준 | 질문 | 점수 |
|---|---|---:|
| 문제 정의 | 해결하려던 문제가 분명한가? | 0-2 |
| 내 역할 | 사용자의 역할과 오너십이 분명한가? | 0-2 |
| 해결 과정 | 판단, 실행, 조율 과정이 설명 가능한가? | 0-2 |
| 결과 | 정량 또는 정성 변화가 있는가? | 0-2 |
| 회고 | 배운 점, 제약, 다음 개선이 있는가? | 0-1 |
| JD 적합성 | 목표 직무와 연결되는 역량이 있는가? | 0-1 |

## 판정

- **8-10점**: 포트폴리오 핵심 프로젝트로 적합
- **5-7점**: 보조 프로젝트로 적합, 부족 정보 보강 필요
- **3-4점**: 마스터 재료에는 보관하되 포트폴리오 노출은 신중
- **0-2점**: 현재 자료로는 포트폴리오 부적합
