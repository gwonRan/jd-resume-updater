# 포트폴리오 증거 추출 프레임워크

## 추출 항목

- 프로젝트명
- 기간, 마감일, 출시일, 운영 기간
- 배경과 문제 상황
- 목표와 성공 기준
- 사용자의 역할
- 오너십 범위
- 주요 의사결정
- 해결 과정
- 사용한 스킬/도구와 활용 맥락
- 협업 대상
- 산출물
- 정량 성과
- 정성 성과
- 실패/제약/트레이드오프
- 회고와 다음 개선점
- JD와 연결되는 역량
- 기밀 또는 일반화 필요 정보
- 부족한 정보

## 신뢰도 라벨

- **확정**: 자료에서 직접 확인됨
- **확인 필요**: 그럴듯하지만 명시되지 않음
- **부족**: 케이스 스터디 품질을 위해 필요한 정보
- **민감**: 익명화 또는 일반화가 필요한 정보
